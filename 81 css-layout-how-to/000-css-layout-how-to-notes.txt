#什么叫做布局？
- 又称版式布局，是网页UI设计师将有限的视觉元素进行有机的排列组合。
-网页设计的特点：继承了报考杂志的设计方式，但具有自身的特点，一是网页的宽度可以按照屏幕的分辨率设计成自适应的，二是网页的高度理论上没有限制（突破了传统纸媒长度的限制）。
- 常见的布局为头+内容+底部，内容再分1列，2列，3列等等

#一列布局
- 一般作为网站首页，例如百度搜索页
- 简单明了，主题突出，适合放置简单的内容，例如图片，大号标题文字，不适合放置多行文字，这是由于一列布局过长，阅读多行文字容易串行。

#两列布局
- width： 20% + float：left
